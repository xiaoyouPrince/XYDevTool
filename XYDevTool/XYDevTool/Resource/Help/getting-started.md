# 快速开始

## 1. 打开主界面

启动 XYDevTool 后，主窗口列出各功能入口，点击对应按钮即可打开独立窗口。

## 2. 常用入口

| 按钮 | 功能 |
|------|------|
| Json2Class | JSON 转 Swift Model |
| JSON 格式化 | 格式化 JSON 文本 |
| 网络请求 | HTTP 调试工具 |
| 创建图标 | AppIcon 生成 |

## 3. 菜单栏 Help

通过 **Help** 菜单可打开完整帮助文档，包含各模块详细说明：

- **⌘ ?** — 打开帮助中心
- 各功能说明 — 直达对应章节
- GitHub / 检查更新 — 外链与发布页

## 4. 建议工作流（接口调试）

1. 打开 **网络请求** 工具
2. 在设置中配置变量（如 `token`、`host`）
3. 填写 URL / Header / Body，使用 `{{key}}` 引用变量
4. 如需签名，在设置中配置 **前置脚本** 并绑定到当前请求
5. Submit 发请求，响应区查看结果
6. 使用 **后置脚本** 从响应中提取 token 写回变量

## 5. 数据备份

网络工具 → 设置 ⚙️ → **导出配置**，会生成多份 JSON（历史、变量、前置/后置脚本），可在另一台机器 **导入配置** 恢复。
