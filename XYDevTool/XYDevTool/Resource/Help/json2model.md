# JSON 转 Model

## 功能

读取 JSON 数据，自动生成 **Swift Class** 模型代码，减少手写 Model 的工作量。

## 主要能力

- 自动解析嵌套的 `array` / `dictionary`，生成独立类型
- 数组元素为字典时，合并出现过的 key，减少测试数据不全导致的字段丢失
- 支持自定义类前缀、继承基类、遵守协议
- 可选：将 JSON value 作为属性注释
- 可选：自动生成 `CodingKeys`

## 使用步骤

1. 主界面进入 **Json2Class**
2. 粘贴或导入 JSON
3. 配置类名前缀、基类等选项
4. 生成代码并复制到工程

## 提示

复杂 JSON 建议先用真实接口返回的完整样例，以便生成更准确的模型结构。
